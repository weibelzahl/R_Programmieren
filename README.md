# R Programmieren

A German swirl course for learning R in R. It is based on the course R Programming in the official swirl repository. 

The course has been translated and adapted to support the statistics course for psychologists at PFH Private University of Applied Sciences Göttingen.

# R Programmieren

Dies ist ein swirl-Kurs um R in R zu lernen. Der Kurs basiert auf dem Kurs R Programming aus dem swirl Repository.

Der Kurs wurde übersetzt und an die Bedürfnisse der Statistikausbildung für Psychologen an der PFH Private Hochschule Göttingen angepasst.
